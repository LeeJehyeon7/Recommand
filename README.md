# 리케이온 교양 과목 추천 시스템

리케이온 유학생을 위한 맞춤형 교양 과목 추천 웹 애플리케이션입니다.

## 📁 프로젝트 구조

```
jinny/
├── index.html         # 메인 HTML 페이지
├── styles.css         # 스타일시트
├── app.js             # JavaScript 로직
├── courses_new.csv    # 과목 데이터 (업데이트 대상) ⭐
└── README.md          # 프로젝트 문서
```

## 🚀 GitHub Pages 배포

이 프로젝트는 `https://leejehyeon7.github.io/Recommand/`에 배포되어 있습니다.

**중요**: `courses.csv` 파일이 깨졌으므로 `courses_new.csv` 파일을 사용합니다.

변경사항을 GitHub Pages에 반영하려면:

1. `courses_new.csv` 파일 업데이트
2. GitHub에 커밋 및 푸시
3. 자동으로 반영됩니다

## 📝 과목 데이터 업데이트 방법

### CSV 파일 형식 (새로운 구조)

`courses_new.csv` 파일은 다음 열(column)로 구성됩니다:

| 열 이름 | 설명 | 예시 |
|---------|------|------|
| `NO` | 과목 번호 | `1` |
| `과목` | 과목 분야 | `문화와 표현` |
| `교수` | 과목명 (수업명) | `리얼리스틱글쓰기` |
| `한국어 요구 수준(종합)` | 전반적 한국어 수준 | `중급A` |
| `듣기` | 듣기 수준 (1-4) | `2` |
| `말하기` | 말하기 수준 (1-4) | `2` |
| `읽기` | 읽기 수준 (1-4) | `2` |
| `쓰기` | 쓰기 수준 (1-4) | `2` |
| `수업자료 사전제공` | 자료 제공 여부 (Y/N) | `Y` |
| `한국문화/역사 정도` | 문화 내용 정도 | `전혀 다루지않음` / `일부 다룸` / `대부분 다룸` |
| `학생간 상호작용` | 상호작용 여부 (Y/N) | `N` |
| `분야` | 교수명 | `윤희진` |

### 과목 추가 예시

```csv
52,인간과 윤리,창의적글쓰기,중급B,3,2,3,3,Y,일부 다룸,Y,김철수
```

### 주의사항

1. **쉼표와 따옴표**: 설명에 쉼표가 포함된 경우 전체를 큰따옴표(`"`)로 감싸세요
2. **한국어 수준**: `초급`, `중급A`, `중급B`, `중고급`, `고급` 중 선택
3. **듣기/말하기/읽기/쓰기**: 1~4 사이의 숫자
4. **Y/N 필드**: 대문자 Y 또는 N만 사용
5. **헤더 유지**: 첫 번째 줄(헤더)은 절대 삭제하지 마세요
6. **인코딩**: 파일은 **UTF-8 인코딩**으로 저장해야 합니다

### Excel에서 편집하기

1. `courses_new.csv`를 Excel에서 열기
2. 과목 정보 수정/추가
3. "다른 이름으로 저장" → 파일 형식: **CSV UTF-8 (쉼표로 분리)(*.csv)** 선택
4. 저장 후 GitHub에 업로드

## 🎯 설문 조사 항목

새로운 설문 조사는 다음 항목들을 포함합니다:

1. **한국어 요구 수준 (종합)**: 초급 ~ 고급
2. **듣기 수준**: 1 (매우 낮음) ~ 4 (높음)
3. **말하기 수준**: 1 (매우 낮음) ~ 4 (매우 높음)
4. **읽기 수준**: 1 (매우 낮음) ~ 4 (매우 높음)
5. **쓰기 수준**: 1 (매우 낮음) ~ 4 (매우 높음)
6. **수업 자료 사전 제공**: 필요 여부 선택
7. **한국 문화/역사 다루는 정도**: 선호도 선택
8. **학생 간 상호작용**: 협업 선호도
9. **관심 분야**: 복수 선택 가능 (문화와 표현, 인간과 윤리, 삶과 건강, 역사와 이론, 사회와 경제)

## 🧮 추천 알고리즘

**가중치 시스템**:
- 한국어 종합 수준 매칭: 10점
- 개별 스킬 매칭 (듣기/말하기/읽기/쓰기 각각): 3점
- 수업 자료 제공 매칭: 2점
- 문화 내용 매칭: 2점
- 학생 상호작용 매칭: 2점
- 관심 분야 매칭: 5점

**매칭 로직**:
- 사용자 한국어 수준보다 2단계 이상 높은 과목은 제외
- 개별 스킬이 요구 수준 이상이면 가산점
- "상관없음" 선택 시 해당 항목 점수 부여
- 최종 점수 기준 상위 10개 과목 추천

## 🐛 문제 해결

### 로컬에서 추천 과목이 표시되지 않아요

**원인**: 로컬 파일 시스템(`file://`)에서는 브라우저 보안 정책(CORS) 때문에 CSV 파일을 로드할 수 없습니다.

**해결됨**: 시스템이 자동으로 로컬 환경을 감지하고 폴백 데이터(샘플 5개)를 사용합니다.

**로컬 테스트 방법**:
1. `index.html`을 브라우저로 직접 열기
2. 5개 샘플 과목으로 기능 테스트
3. 브라우저 콘솔(F12)에서 "로컬 환경 감지" 메시지 확인

**전체 데이터 테스트 방법** (선택):
```bash
# Python 3가 설치되어 있는 경우
python -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

### courses.csv 파일이 깨졌어요

**해결책**: `courses_new.csv` 파일을 사용합니다.

- `courses.csv`는 인코딩 문제로 깨짐
- `courses_new.csv`가 올바른 UTF-8 인코딩 파일
- `app.js`가 자동으로 `courses_new.csv`를 로드합니다

### 스타일이 적용되지 않아요

- `index.html`, `styles.css`, `app.js`, `courses_new.csv`가 같은 폴더에 있는지 확인
- 브라우저 캐시 삭제 후 새로고침 (Ctrl + Shift + R)
